# Organization Type Registry

This file is the authoritative catalog of CV format references keyed to organization
type and maturity. Read this file when identifying the appropriate CV template for
the target company. Load the corresponding template file in Phase 4a as instructed
by the skill.

---

## Large Enterprise Established

**Use when:** The target company is an established pharma, large CRO, or mature
biotech. Indicators: stable operating model, scaled infrastructure, established
regulatory and governance posture.

**Do NOT use when:** The company is in a growth or transformation state -- size
alone does not determine this selection. A large company undergoing significant
transformation belongs in Mid-Size Scale-Up.

**Template file:** `knowledge/CV_General_Lrge_Enterprise_Established.md`

---

## Mid-Size Scale-Up

**Use when:** The target company is growth-stage biotech, OR any company undergoing
significant transformation regardless of current size or maturity. The signal is
organizational state, not headcount.

**Do NOT use when:** The company is stable, scaled, and operating in steady state.

**Template file:** `knowledge/CV_General_Mid_Size_Scale_Up.md`
