# Contact Information

This file is the single source of truth for all personal contact and profile information used in CV generation. Load this file in Phase 5a before generating any CV document. Use these values for the contact line; do not hardcode contact information in generated scripts.

---

## Fields

**Name:** [Your Name]

**Email:** [Your Email]

**Phone:** [Your Phone]

**LinkedIn:** [Your LinkedIn URL]

**GitHub:** [Your GitHub URL]

**Website:** [Your Website URL, or remove this line if none]

---

## Usage Notes

- All fields except Website are required in the CV contact line
- Website is optional; include only if a value is present
- GitHub URL should appear in the contact line alongside LinkedIn
- If additional profile URLs are added in the future, include them here and update the CV generation skills to reference them
